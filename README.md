# Multi-Agent Research Assistant with MCP Integration

## Overview
This project is a **Multi-Agent Research Assistant** built using:

- Django REST Framework (Backend API)
- LangGraph (Agent orchestration)
- MCP (Tool integration)
- Streamlit (Frontend UI)

---

## Architecture

User → Streamlit UI → Django API → LangGraph Agents → Database → Response

---

## Features

- Multi-agent system (Planner + Researcher)
- REST APIs (POST, GET, PUT, DELETE)
- Conversation history storage
- Streamlit UI
- MCP tool integration
